## 📱 프로젝트 개요

### 서비스 소개
**관상네컷** - 네컷 사진으로 보는 나의 관상 분석과 모임 궁합

### MVP 핵심 기능
1. **개인 관상 분석**: 얼굴 특징점 추출 → 부위별 긍정 해석 → 미래 얼굴 네컷 생성
2. **모임 궁합 분석**: 2~6명 구성 → 관계 네트워크 시각화 → 체질 기반 추천
3. **결과 공유**: 네컷 프레임 생성 → SNS 공유 → PNG 다운로드

## 🔬 기술 스택 조사

### 핵심 기술 선정

| 분야 | 기술 | 선정 이유 | 우선순위 |
|------|------|-----------|----------|
| **얼굴 인식** | MediaPipe | 468개 랜드마크, 실시간 처리 가능 | ⭐⭐⭐ |

## 🤔 주요 검토 사항

### 1. 미래 얼굴 생성 방식
- **StyleGAN3**: 고품질이지만 GPU 필수, 처리 시간 3~5초
- **Face Aging 모델**: 빠르지만 품질 낮음
- **결정**: StyleGAN3 우선 검토, 처리 시간 개선 방안 연구 필요

### 2. 관상 해석 비용 절감
- GPT-4 API 비용: 요청당 약 $0.02
- 해결 방안: 자주 나오는 패턴 템플릿화, 프롬프트 최적화

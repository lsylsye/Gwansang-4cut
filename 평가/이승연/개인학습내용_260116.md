

## 학습한 기술 스택
React

## 학습 내용
- React 강의를 수강하며 컴포넌트 기반 UI 구조와 props/state 흐름을 학습했습니다.  
- useState/useEffect/useRef를 활용해 상태 관리, 사이드 이펙트 처리, DOM 제어 패턴을 익혔습니다.  
- 배열 메서드(map/filter)와 비동기 흐름, 클린업(정리 함수) 개념을 실습으로 적용해봤습니다.  
- 또한 컴포넌트 구조 설계(상태 위치 선정, 부모-자식 데이터/이벤트 전달) 관점을 정리했습니다.  


## 아이디어 정리

<기획 회의에서 제안했던 아이디어 정리>
- 점메추다이스: 위치 기반으로 주변 식당을 수집·필터링한 뒤 주사위(가중 랜덤) 방식으로 점심 메뉴를 추천하는 간편 플랫폼을 제안하였습니다. 메뉴 결정 피로를 줄이고, 거리·평점 등 근거가 있는 “납득 가능한 랜덤 추천”을 목표로 설정하였습니다.

- 한컷식단: 음식 사진 한 장만 촬영하면 AI가 음식 종류를 분류하고 대략적인 칼로리를 추정하는 식단 기록 플랫폼을 구상하였습니다. 입력 부담을 최소화하고, 주간 리포트와 목표 관리까지 확장 가능한 “식단 코치” 형태를 목표로 하였습니다.

- 커리어픽: 사진 1장으로 취업용 프로필 사진을 생성·가이드하고, 동시에 개인 조건에 맞는 취업 지원 제도와 복지를 큐레이션해주는 플랫폼을 제안하였습니다. 이미지 생성 AI와 복지 정보 RAG를 결합한 구조를 특징으로 하였습니다.

- 실시간 웨이팅 공유 SNS: 음식점·카페·전시 등의 대기 시간을 사용자가 직접 실시간으로 공유하는 플랫폼을 아이디어로 제시하였습니다. 기존 지도 리뷰와 차별화되는 ‘실시간성’을 핵심 가치로 설정하였습니다.

- 복지·안전·생활 SNS 아이디어들: 전세사기 예방, 중고거래 사기 경보, 복지 길찾기, 고령층 디지털 격차 해소, 재난·안전 정보 공유 등 사회 문제를 해결하는 SNS 기반 아이디어들을 다수 제안하고 비교 검토하였습니다.

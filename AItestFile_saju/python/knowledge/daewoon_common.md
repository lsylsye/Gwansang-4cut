# 대운(大運) 공통 문서

1번 블록(인생 회고)과 2번 블록(취업운)에서 공통으로 참고하는 대운 계산·해석 기준입니다.

---

## 시간 로직 (필수)

- **입력 생년월일시**: **서울시간(Asia/Seoul)** 기준으로 해석합니다.
- 서버 내부 계산(절기, 대운 시작 나이)도 **Asia/Seoul** 기준으로 통일합니다.
- 절기 시각: 현재는 해당 연도·해당일 **12:00 KST** 근사 사용. (향후 천문 시각으로 교체 권장)

---

## 대운 계산 기준

- **기준**: **월주**(태어난 달의 간지). 연주 아님.
- **순행/역행**: 연간(年干) 음양 + 성별  
  - 양년생 남자 / 음년생 여자 → **순행**  
  - 음년생 남자 / 양년생 여자 → **역행**  
  - 연간 양: 갑·병·무·경·임 / 음: 을·정·기·신·계
- **시작 나이**: 출생 시각 ~ 다음(순행) 또는 이전(역행) **절기**까지 시간 차이 → **3일 = 1세** 환산, **소수점 유지** (예: 7.6세).
- **대운 간지**: 월주에서 60갑자로 순행/역행하며 **10년 단위**로 나열. 월주 계산과 **동일한 절기 데이터** 사용.

---

## 대운 → 체감 서사 규칙 (1번·2번 블록 참고)

- 대운 **단독 길흉**이 아니라 **일주가 그 운을 어떻게 체감했는가**만 본다.
- **선택폭 체감**: 비견·겁재·식상 → 넓음, 재성 → 조건부, 관성 → 좁음, 인성 → 멈춤.
- **기본 태도**: 일주 × 대운 오행 반응으로 "더 참는다", "계산하며 움직인다" 등 **태도**만 서술.
- **반복 감정**: 넓음→흥분/조급, 조건부→긴장, 좁음→피로/답답, 멈춤→무기력.
- **시점 보정**: 대운 시작 나이 ±0.5~1.5년 구간을 "그 시기 전후"로 서술 가능.

---

## API·구현 체크리스트

- 절기 = 시각(타임스탬프)
- 월주 = 대운 출발점
- 순/역 = 연간 음양 + 성별
- 출생 시각 필수(분 단위)
- 대운 시작 나이 소수점 유지
- 타임존 통일(Asia/Seoul)
- 계산 기준 명시(3일=1세, 절기 근사 등)

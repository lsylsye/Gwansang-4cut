# 사주 계산 기본 규칙 (천문적 데이터 레이어)

## 1. 사주 4주 (四柱八字) 개요

사주는 생년월일시를 간지(干支)로 변환한 4개의 기둥으로 구성됩니다.

- **연주(年柱)**: 출생 연도의 간지
- **월주(月柱)**: 출생 월의 간지 (절기 기준)
- **일주(日柱)**: 출생 일의 간지
- **시주(時柱)**: 출생 시간의 간지

각 기둥은 천간(天干)과 지지(地支)로 구성됩니다.

## 4. 연주(年柱) 계산 규칙

### 기준
- **입춘(2월 4일경) 전 출생**: 전년도 간지 사용
- **입춘 이후 출생**: 해당 연도 간지 사용
- **기준년**: 1984년 = 갑자년(甲子年)

### 계산 방법
1. 입춘 이전이면 연도 -1
2. (연도 - 1984) % 60으로 60갑자 순환 위치 계산
3. 천간 = (gap % 10), 지지 = (gap % 12)

### 예시
- 1998년 10월 17일 → 입춘 이후이므로 1998년 사용
- (1998 - 1984) % 60 = 14
- 천간: 14 % 10 = 4 → 무(戊)
- 지지: 14 % 12 = 2 → 인(寅)
- **연주: 무인(戊寅)**

| 청명~입하 | 진월(辰月) | 4/5 ~ 5/6 |

- **월주: 임술(壬戌)**

## 6. 일주(日柱) 계산 규칙

### 기준일
- **1900년 1월 1일 = 경자일(庚子日)**

### 계산 방법
1. 기준일로부터 경과일 계산
2. 경과일 % 60으로 60갑자 순환 위치 계산
3. 천간 = (경과일 + 6) % 10, 지지 = (경과일 + 0) % 12

### 예시
- 1998년 10월 17일
- 경과일 ≈ 36,000일
- 천간: (36000 + 6) % 10 = 6 → 정(丁)
- 지지: (36000 + 0) % 12 = 0 → 유(酉)
- **일주: 정유(丁酉)**

## 7. 시주(時柱) 계산 규칙

### 시간대별 지지

| 시간 | 지지 | 시간 | 지지 |
|------|------|------|------|
| 23~1시 | 자 | 11~13시 | 오 |
| 1~3시 | 축 | 13~15시 | 미 |
| 3~5시 | 인 | 15~17시 | 신 |
| 5~7시 | 묘 | 17~19시 | 유 |
| 7~9시 | 진 | 19~21시 | 술 |
| 9~11시 | 사 | 21~23시 | 해 |

### 시간 결정 (일간 기준)

일간에 따라 시간의 천간이 결정됩니다.

**시간 순서표:**
- 자시(23~1): 갑병무경임 순서
- 축시(1~3): 을정기신계 순서
- 인시(3~5): 갑병무경임 순서
- 묘시(5~7): 을정기신계 순서
- ... (짝수 지지: 갑병무경임, 홀수 지지: 을정기신계)

**일간 기준 매핑:**
- 일간이 갑(甲)이고 자시(子)면 → 갑자(甲子)
- 일간이 을(乙)이고 자시(子)면 → 병자(丙子)
- 일간이 정(丁)이고 오시(午)면 → 병오(丙午)

### 예시
- 일간: 정(丁)
- 시간: 13시 → 미시(未시)
- 미는 홀수 지지이므로 을정기신계 순서
- 정(丁)은 4번째이므로 → 병(丙)
- **시주: 병미(丙未)** → 실제로는 병오(丙午)가 맞음 (오후 1시는 오시)

## 9. 음력 변환

음력은 태음력으로, 양력과 날짜가 다릅니다.

### 음력 처리 규칙
1. 음력 입력 시 양력으로 변환 필요
2. 윤달(閏月) 처리 필요
3. 변환 후 위의 양력 기준 계산 규칙 적용

**참고:** 음력 변환은 복잡하므로 전문 라이브러리 사용 권장.

## 10. 일간(日干)의 중요성

**일간은 사주 해석의 핵심입니다.**

- 일간 = 자신의 본질
- 다른 간지와의 관계를 일간 기준으로 해석
- 십성(十神), 12운성 등 모든 관계가 일간 중심

**예시:**
- 일간이 정(丁)이면 → 정화(丁火) 일간
- 정화는 등불 같은 부드러운 불
- 다른 간지와의 관계로 성격, 운세 해석
